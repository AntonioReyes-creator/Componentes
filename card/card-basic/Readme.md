Tarjeta (card) interactiva usando HTML y CSS puro.
La tarjeta debe girar al pasar el cursor (hover), mostrando una animación suave de rotación 3D.

En la cara frontal, muestra una imagen o título.

En la cara trasera, muestra información adicional o un texto descriptivo.
